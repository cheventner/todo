Methods

find(query, callback)

Source:
  • store.js, line 29
Recherche des éléments en fonction d'une requête donnée sous la forme d'un objet JS.
Example
db.find({foo: 'bar', hello: 'world'}, function (data) {
	 // data will return any items that have foo: bar and
	 // hello: world in their properties
});
Parameters:
Name Type Description
query object The query to match against (i.e. {foo: 'bar'})
callback function The callback to fire when the query has completed running
