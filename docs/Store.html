find(query, callback)

Source:
  • store.js, line 27
Recherche des éléments en fonction d'une requête donnée sous la forme d'un objet JS.
Example
db.find({foo: 'bar', hello: 'world'}, function (data) {
	 data will return any items that have foo: bar and
	 hello: world in their properties
});
Parameters:
Name Type Description
query object La requête à comparer (i.e. {foo: 'bar'})
callback function Le callback à lancer lorsque la requête a terminé l'exécution.
